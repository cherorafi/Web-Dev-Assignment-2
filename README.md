# Web-Dev-Assignment-2

## This assignment recreates 10 native javascript methods
1. map() -> myMap()
2. filter() -> myFilter()
3. some() -> mySome()
4. every() -> myEvery()
5. reduce() -> myReduce()
6. includes() -> myIncludes()
7. indexOf() -> myIndexOf() 
8. lastIndexOf() -> myLastIndexOf()
9. Object.keys() -> Object.myKeys()
10. Object.values() -> Object.myValues()
